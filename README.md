📌 AI-Powered KQL Query Generator
🚀 Overview

The AI-Powered KQL Query Generator is a modern tool designed to simplify and accelerate the process of writing Kusto Query Language (KQL) queries using natural language input.

Security analysts, SOC engineers, and IT professionals often spend significant time crafting queries manually. This tool reduces that effort by leveraging AI/LLM capabilities to automatically generate relevant, structured, and usable KQL queries.

The tool is especially useful in environments such as:

Microsoft Sentinel
Azure Log Analytics
Defender XDR
SIEM and SOC platforms

It bridges the gap between human intent and technical query syntax, making detection engineering faster and more accessible.

🎯 Purpose

The main goal of this tool is to:

Reduce time spent writing KQL queries
Improve efficiency in SOC operations
Enable faster threat hunting and detection creation
Assist beginners in learning KQL
Standardize query generation using AI

🧠 Key Features

Natural Language to KQL conversion
AI-powered query generation
Supports multiple log sources
Easily customizable queries
Model-agnostic LLM integration
Lightweight and flexible architecture
Easy to deploy and extend
SOC-friendly design
Beginner-friendly interface
Fast response generation
